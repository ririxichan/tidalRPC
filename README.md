# Unofficial Tidal Rich Presence

## Features

- Single executable binary.
- Cross-platform (Windows, macOS).
- Super fast.
- Tray for closing Application.
- Coded in TypeScript.

## Supported OS

- Windows
- macOS (High Sierra+)
